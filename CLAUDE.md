# nnotify

Animated toast notifications for Nette — replaces flash messages.

## Key Classes
- `ToastControl` — Nette UI component rendering Bootstrap toasts
- `ToastControlFactory` — DI factory
- DI Extension: `NNotifyExtension`

## Conventions
- PHP 8.2+ with `declare(strict_types=1)`
- PSR-4 autoloading in `src/`
- Nette DI Extension for service registration
- PHPStan level 9
- Part of `jansuchanek/*` on Packagist
- GitHub: https://github.com/JanSuchanek/nnotify
